================================================================================
                    HMS PRO - HOSPITAL MANAGEMENT SYSTEM
                         PROJECT DOCUMENTATION
================================================================================

PROJECT TYPE: React + Vite Application
LANGUAGE: TypeScript
VERSION: 0.1.0

================================================================================
TECHNOLOGY STACK
================================================================================

Frontend Framework:
  - React 18.3.1
  - Vite 5.2.0 (Build Tool)
  - TypeScript 5.x

Styling:
  - Tailwind CSS v4.1.9
  - Shadcn UI (Radix UI Components)
  - Custom CSS Variables

UI Components:
  - 57 Shadcn UI Components (Button, Card, Dialog, etc.)
  - Lucide React Icons
  - Recharts (Charts Library)

Utilities:
  - date-fns (Date formatting)
  - clsx & tailwind-merge (Class utilities)
  - React Router DOM 6.22.3 (Routing)

================================================================================
IMPLEMENTED FEATURES
================================================================================

1. DASHBOARD
   - Real-time statistics (Total Patients, Outdoor, Indoor, Active Cases)
   - Interactive charts (Area, Pie, Bar, Line)
   - Recent patients list with status
   - Today's appointments timeline
   - Quick action buttons

2. NAVIGATION SYSTEM
   - Collapsible sidebar with animations
   - Real-time clock in navbar (Date & Time)
   - Nested menu items with tree view
   - Active state indicators
   - Responsive design

3. MASTERS MODULE (Sub-menus)
   - Departments Management
   - Consultants Registry
   - Employees Management
   - Items Catalog
   - Vendors Directory
   - Surgery Records
   - Service/Tests Management
   - Beds Allocation

4. UI/UX ENHANCEMENTS
   - Premium gradient backgrounds
   - Smooth hover effects with color transitions
   - Collapsible animations using Radix UI
   - High contrast text (Deep Black)
   - Tree-view style for hierarchical items
